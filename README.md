# Odoo_Repo
HRMS – AI-Powered Human Resource Management System

A modern, AI-driven Human Resource Management System built to simplify recruitment, employee management, attendance, payroll, performance tracking, and workplace communication through an intuitive dashboard.

⸻

Overview

HRMS is a next-generation Human Resource Management System designed to centralize every HR operation into a single platform. Instead of managing multiple spreadsheets and disconnected tools, HR teams can recruit, onboard, monitor, evaluate, and engage employees from one unified workspace.

The platform combines modern UI with intelligent automation to improve productivity, reduce manual work, and provide actionable insights for organizations.
